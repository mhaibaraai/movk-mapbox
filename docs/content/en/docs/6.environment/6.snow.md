## Introduction

`MapboxSnow` calls `setSnow` to add a snow effect to the map. The `options` prop is forwarded directly to the Mapbox snow specification (`density`, `intensity`, `color`, etc.). Omit it to use the default snow effect.

::note{color="warning"}
`setSnow` is an experimental Mapbox API that requires a v3 style (such as `standard`). It may change in future versions.
::
