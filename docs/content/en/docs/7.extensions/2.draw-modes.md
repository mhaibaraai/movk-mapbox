## Introduction

`movkDrawModes` is a collection of custom draw modes provided by movk (circle / ellipse / rectangle / sector). Merge it with `MapboxDraw.modes` and pass the result to [MapboxDrawControl](/docs/extensions/draw) via `options.modes`, then switch between modes using `changeMode`.

Auto-imported from `@movk/mapbox` — no manual import needed.

## API

### `movkDrawModes`{lang="ts-type"}

A mode collection object where each key is a mode name (passed to `changeMode`) and each value is a mapbox-gl-draw mode object. Individual modes can also be imported by name (`drawRectangleMode` / `drawCircleMode` / `drawEllipseMode` / `drawSectorMode`).

::field-group
  ::field{name="draw_rectangle" type="DrawCustomMode"}
  Rectangle draw mode.
  ::
  ::field{name="draw_circle" type="DrawCustomMode"}
  Circle draw mode.
  ::
  ::field{name="draw_ellipse" type="DrawCustomMode"}
  Ellipse draw mode.
  ::
  ::field{name="draw_sector" type="DrawCustomMode"}
  Sector draw mode.
  ::
::
