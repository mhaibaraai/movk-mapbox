## Introduction

`MapboxDrawControl` provides declarative drawing on top of [mapbox-gl-draw](https://github.com/mapbox/mapbox-gl-draw): `v-model:features` controls the feature collection (assigning imports, drawing writes back), `v-model:mode` controls the current mode; it emits create / update / delete / selectionchange / modechange events; and exposes imperative methods via `defineExpose`. Child components can inject the instance with [useMapboxDraw](/docs/composables/use-mapbox-draw).
