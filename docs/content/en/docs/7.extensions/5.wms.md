## Introduction

`MapboxWmsLayer` integrates a standard WMS (GetMap) service as a raster layer: `url` and `layers` are required; `version` / `format` / `crs` / `transparent` and other parameters are forwarded as-is; `params` appends custom query parameters. Tiles are requested per-tile using `{bbox-epsg-3857}`.
