## Introduction

`MapboxWmtsLayer` integrates a standard WMTS (KVP GetTile) service as a raster layer: `url` and `layer` are required; `tileMatrixSet` / `style` / `format` are forwarded as-is; when `url` contains `{s}`, `subdomains` is used to expand multi-host URLs; `params` appends auth parameters (e.g. `tk`). [MapboxTiandituLayer](/docs/extensions/tianditu) is built on top of this component.
