## Introduction

`useMapboxCamera` wraps common camera operations: smooth fly-to, easing, instant jump, and fit-to-bounds. All methods return a `Promise` and internally await the map's loaded state before executing, so there is no need to manually check readiness. `fitBounds` additionally accepts any GeoJSON and automatically computes its bounding box.

::note
When used outside a `<MapboxMap>` subtree, specify the target map via `options.mapId`.
::

## API

### `useMapboxCamera()`{lang="ts-type"}

Creates a camera helper.

::field-group
  ::field{name="options.mapId" type="string"}
  Target map id; required when used outside a `<MapboxMap>` subtree.
  ::
::

Returns `UseMapboxCameraReturn`:

::field-group
  ::field{name="flyTo" type="(options: EasingOptions) => Promise<void>"}
  Smoothly flies to the target camera position.
  ::
  ::field{name="easeTo" type="(options: EasingOptions) => Promise<void>"}
  Eases to the target camera position.
  ::
  ::field{name="jumpTo" type="(options: CameraOptions) => Promise<void>"}
  Immediately jumps to the target camera position.
  ::
  ::field{name="fitBounds" type="(target: LngLatBoundsLike | GeoJSON, options?: FitBoundsOptions) => Promise<void>"}
  Fits the viewport to the target bounds; `target` accepts a bounds object or any GeoJSON (bounding box is computed automatically).
  ::
::
