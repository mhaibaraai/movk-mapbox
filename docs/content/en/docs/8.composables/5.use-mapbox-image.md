## Introduction

`useMapboxImage` loads a remote image and registers it as a style image, making it available for symbol layers via `icon-image`. After `setStyle` clears all style images, it is automatically re-added via `onReady`; it is removed when the component unmounts. The returned `loaded` state lets you show placeholder feedback while the image is loading.

::note
Set `sdf: true` to register the image as an SDF icon, which can then be tinted via `icon-color`. When used outside a `<MapboxMap>` subtree, specify the target map via `options.mapId`.
::

## API

### `useMapboxImage()`{lang="ts-type"}

Registers a named image.

::field-group
  ::field{name="name" type="string" required}
  The name to register the image under, used as the `icon-image` value.
  ::
  ::field{name="url" type="string" required}
  URL of the image to load.
  ::
  ::field{name="options.mapId" type="string"}
  Target map id; required when used outside a `<MapboxMap>` subtree.
  ::
  ::field{name="options.sdf" type="boolean"}
  Register as an SDF icon (tintable via `icon-color`).
  ::
  ::field{name="options.pixelRatio" type="number"}
  Pixel density ratio, defaults to `1`.
  ::
::

Returns `{ loaded: Ref<boolean> }`: whether the image has finished loading.
