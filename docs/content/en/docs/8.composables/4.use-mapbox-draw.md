## Introduction

`useMapboxDraw` injects the [MapboxDraw](https://github.com/mapbox/mapbox-gl-draw) instance (wrapped in a `ShallowRef`) provided by the nearest `<MapboxDrawControl>`, enabling child components to programmatically switch draw modes, add or remove features, and more.

::note
Must be called inside a `<MapboxDrawControl>` subtree, otherwise it will throw. The instance is assigned after the control mounts, so check for a defined value before use.
::

## API

### `useMapboxDraw()`{lang="ts-type"}

Injects the current draw instance. No parameters.

Returns `ShallowRef<MapboxDraw | undefined>`: the draw instance from the nearest `MapboxDrawControl`; `undefined` before the control mounts.
