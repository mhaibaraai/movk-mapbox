## Introduction

`MapboxTooltip` is a hover mode for popups: it appears and follows the cursor when the mouse is over a feature in the `layerId` layer, and disappears when the cursor leaves. The default slot exposes the hit feature (`feature`) so you can render its properties. Event delegation is wired at the map level, so the target layer can be mounted after this component and will still be detected.
