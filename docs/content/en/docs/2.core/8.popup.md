## Introduction

`MapboxPopup` anchors a popup at the specified `lnglat`, rendering arbitrary content via the default slot. The position updates reactively when `lnglat` changes, and a `close` event is emitted when the popup is dismissed.

::note
`MapboxTooltip` holds a single Popup instance and cannot show several feature popups at once — its `feature` comes from mouse events, and the component does not hold the feature list. To keep layer-feature popups open simultaneously, iterate the source array that builds your GeoJSON and render one `MapboxPopup` per point. If your points are DOM markers, [`MapboxMarker`](/docs/core/marker) with `:open="true"` is more direct.
::
