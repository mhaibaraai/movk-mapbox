## Introduction

`MapboxLottieMarker` renders a Lottie vector animation at a marker position using lottie-web (built on [MapboxMarker](/docs/core/marker)'s DOM element, not icon-frame driven). The animation source is either `animationData` (inline JSON) or `path` (remote URL) — pick one. Position is bound via `v-model:lnglat`.
