## Introduction

`MapboxLayer` is the general-purpose layer component: `type` selects the render mode, `paint` / `layout` / `filter` control styling, and reactive changes are applied incrementally via `setPaintProperty` / `setLayoutProperty` / `setFilter` — the layer is never rebuilt. `source` accepts either a `sourceId` string referencing a [MapboxSource](/docs/core/source) or an inline source object (which creates an anonymous source automatically and removes it when the layer unmounts). `beforeId` controls insertion order.
