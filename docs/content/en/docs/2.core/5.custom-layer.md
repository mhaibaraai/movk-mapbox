## Introduction

`MapboxCustomLayer` is the escape hatch for `CustomLayerInterface`: it wires your custom WebGL layer (with its own `onAdd` / `render` callbacks) into the component lifecycle — calling `addLayer` when the map is ready, removing it on unmount, and rebuilding via `onReady` after a basemap switch. Suitable for deep custom rendering with Three.js, deck.gl, or raw WebGL.
