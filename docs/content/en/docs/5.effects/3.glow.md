## Introduction

`MapboxGlowCircle` renders a glowing circle on point features: a solid core with a blurred halo. Enable `pulse` for a breathing pulse animation; `color` / `radius` / `glowScale` / `duration` control the appearance and timing.

## Usage

Switch the primary color to preview the glowing pulse:

::component-example
---
name: GlowExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#f59e0b'
    items:
      - '#f59e0b'
      - '#3b82f6'
      - '#10b981'
      - '#f43f5e'
---
::
