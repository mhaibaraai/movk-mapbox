## Introduction

`MapboxRadar` renders a radar sweep on point features: a canvas-generated sector image is used as a symbol icon and rotated per frame. `color` / `radius` / `speed` control the color, sweep radius, and rotation speed.

## Usage

Adjust `speed` (degrees per second) to observe the sweep rotation:

::component-example
---
name: RadarExample
prettier: true
clientOnly: true
options:
  - name: speed
    label: speed
    default: 120
    items:
      - 60
      - 120
      - 240
---
::
