## Introduction

`MapboxDiffusionCircle` renders a diffusion circle on point features: multiple concentric rings periodically expand outward and fade; `color` / `maxRadius` / `duration` / `rings` / `opacity` control the appearance and timing.

## Usage

Switch the primary color to preview the diffusion animation:

::component-example
---
name: DiffusionExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#3b82f6'
    items:
      - '#3b82f6'
      - '#10b981'
      - '#f59e0b'
      - '#f43f5e'
---
::
