## Introduction

`MapboxWaveCircle` renders a wave circle on point features: a persistent solid base dot with periodically expanding stroked ripples; `color` / `baseRadius` / `maxRadius` / `duration` / `rings` control the appearance and timing.

## Usage

Switch the primary color to preview the ripple expansion:

::component-example
---
name: WaveExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#10b981'
    items:
      - '#10b981'
      - '#3b82f6'
      - '#f59e0b'
      - '#f43f5e'
---
::
