## Introduction

`MapboxMigration` generates bezier arcs from origin-destination pairs (`data` is a `{ from, to }` list), with particles looping along the arcs and leaving gradient trails; `color` / `curvature` / `duration` / `lineWidth` / `particleRadius` control the appearance and timing.

## Usage

Switch the primary color to preview the migration routes departing from Beijing:

::component-example
---
name: MigrationExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#f43f5e'
    items:
      - '#f43f5e'
      - '#3b82f6'
      - '#10b981'
      - '#f59e0b'
---
::
