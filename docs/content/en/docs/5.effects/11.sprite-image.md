## Introduction

`MapboxSpriteImage` slices a sprite sheet (row-major multi-frame layout) into individual frames and registers them as a frame-driven icon, looped per frame on a symbol layer with no decoding dependencies; `frames` / `columns` / `rows` / `frameWidth` / `frameHeight` / `fps` / `size` control the slicing and playback.
