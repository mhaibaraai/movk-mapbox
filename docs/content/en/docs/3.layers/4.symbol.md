## Introduction

Set `type="symbol"` on `MapboxLayer` to render text and icon labels. `text-field` reads feature properties via `['get', ...]`; layout properties like `text-size` and `text-offset` control placement. Icons must be registered first via [useMapboxImage](/docs/composables/use-mapbox-image).

::note{to="/docs/core/layer"}
Symbol layers are powered by the generic MapboxLayer component. See its documentation for full Props and events reference.
::
