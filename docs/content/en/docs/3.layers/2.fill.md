## Introduction

Set `type="fill"` on `MapboxLayer` to render a polygon layer. `fill-color` and `fill-opacity` control the style; combine with `['get', ...]` expressions to color each feature individually.

::note{to="/docs/core/layer"}
Fill layers are powered by the generic MapboxLayer component. See its documentation for full Props and events reference.
::
