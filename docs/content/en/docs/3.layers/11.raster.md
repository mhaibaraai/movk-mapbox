## Introduction

`MapboxRasterLayer` overlays an external raster tile service using a `{z}/{x}/{y}` template URL. Supports `tileSize`, `scheme`, `minzoom`, `maxzoom`, `attribution`, and `opacity`. For Chinese basemap services like Tianditu, see [Extensions](/docs/extensions/tianditu).
