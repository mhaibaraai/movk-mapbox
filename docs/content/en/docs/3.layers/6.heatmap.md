## Introduction

Set `type="heatmap"` on `MapboxLayer` to render point density as a heatmap. `heatmap-radius`, `heatmap-intensity`, and `heatmap-opacity` control the visual appearance. For property-weighted temperature heatmaps, see [MapboxTemperature](/docs/environment/temperature).

::note{to="/docs/core/layer"}
Heatmap layers are powered by the generic MapboxLayer component. See its documentation for full Props and events reference.
::
