## Introduction

`MapboxClusterLayer` automatically clusters large point datasets. Internally it uses a cluster-enabled GeoJSON source to render three layers: cluster circles, count labels, and unclustered points. Clicking a cluster zooms to the expand level by default (`autoExpand`), and emits `clusterClick` / `pointClick` events.
