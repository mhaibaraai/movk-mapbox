## Introduction

Set `type="line"` on `MapboxLayer` to render a line layer. `line-width` and `line-color` control the stroke; `line-cap` and `line-join` control end-point and corner styles.

::note{to="/docs/core/layer"}
Line layers are powered by the generic MapboxLayer component. See its documentation for full Props and events reference.
::
