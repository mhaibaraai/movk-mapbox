## Introduction

`formatDistance` / `formatArea` format raw meter / square-meter values into human-readable strings with units: distances below one kilometer are rounded to whole meters, above that are shown with two decimal places in km; areas below one square kilometer are rounded to whole square meters, above that are shown with two decimal places in km². [useMeasure](/docs/composables/use-measure) uses these functions to format its live readings.

## API

### `formatDistance()`{lang="ts-type"}

Format a distance value.

::field-group
  ::field{name="meters" type="number" required}
  Distance value in meters.
  ::
::

Returns `string`: whole meters below one kilometer (e.g. `850 m`), two decimal places in km above (e.g. `1.23 km`).

### `formatArea()`{lang="ts-type"}

Format an area value.

::field-group
  ::field{name="squareMeters" type="number" required}
  Area value in square meters.
  ::
::

Returns `string`: whole square meters below one square kilometer (e.g. `5000 m²`), two decimal places in km² above (e.g. `1.20 km²`).
