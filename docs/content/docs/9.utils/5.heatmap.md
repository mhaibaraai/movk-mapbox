## 简介

`heatmapPaint` 一步生成 heatmap 图层的完整 paint：把指定要素属性按 `weightRange` 线性映射为 0..1 热力权重，并按 `heatmap-density` 经颜色断点着色（density 0 处自动透明，避免全图铺底色）。免去手写冗长的 `interpolate` 表达式。

从 `@movk/mapbox/utils/heatmap` 导入（非自动导入）。

## API

### `heatmapPaint()`{lang="ts-type"}

生成 heatmap 图层 paint。

::field-group
  ::field{name="options.weightProperty" type="string"}
  权重取值的要素属性，默认 `'temperature'`。
  ::
  ::field{name="options.weightRange" type="[number, number]"}
  权重属性取值范围 `[min, max]`，线性映射到 0..1，默认 `[0, 40]`。
  ::
  ::field{name="options.colorStops" type="[number, string][]"}
  密度-颜色断点 `[密度比 0..1, 颜色][]`；缺省为蓝→红五档，density 0 处自动透明。
  ::
  ::field{name="options.radius" type="number"}
  热力半径（像素），默认 `30`。
  ::
  ::field{name="options.intensity" type="number"}
  热力强度，默认 `1`。
  ::
  ::field{name="options.opacity" type="number"}
  整体不透明度，默认 `1`。
  ::
::

返回热力图层 paint 对象，可直接绑定到 `<MapboxLayer type="heatmap">` 的 `paint`。
