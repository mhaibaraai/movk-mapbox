## 简介

`MapboxWmsLayer` 把标准 WMS（GetMap）服务接入为栅格图层：`url` 与 `layers` 必填，`version` / `format` / `crs` / `transparent` 等透传，`params` 追加自定义查询参数。瓦片按 `{bbox-epsg-3857}` 逐块请求。
