## 简介

`MapboxTiandituLayer` 以天地图 WMTS 瓦片作底图：`layer` 选类型（`vec` 矢量 / `img` 影像 / `ter` 地形，及注记 `cva`/`cia`/`cta`），`annotation` 叠加对应注记层，`tk` 缺省时回退运行时配置的 `tiandituToken`。

::note
天地图坐标系为 WGS84（CGCS2000），自有 WGS84 数据可直接叠加，无需 [坐标转换](/docs/extensions/coordinate)；只有高德 / 腾讯（GCJ02）、百度（BD09）的数据叠加时才需要先转换。
::

## 用法

切换 `layer` 查看矢量/影像/地形底图（均叠加注记）：

::component-example
---
name: TiandituLayerExample
prettier: true
options:
  - name: layer
    label: layer
    default: vec
    items:
      - vec
      - img
      - ter
  - name: annotation
    label: annotation
    default: true
    items:
      - true
      - false
---
::
