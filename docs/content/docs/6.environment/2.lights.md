## 简介

`MapboxLights` 调用 `setLights` 设置地图的 3D 光照列表（`ambient` 环境光 / `directional` 平行光 / `flat` 平面光），作用于 3D 建筑与地形。需配合含 3D 内容的样式（如 `standard`）与倾斜视角。
