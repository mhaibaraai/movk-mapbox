## 简介

`MapboxRain` 调用 `setRain` 为地图添加降雨效果，`options` 透传 mapbox rain 规格（`density`/`intensity`/`color` 等），省略用默认雨效。

::note{color="warning"}
`setRain` 为 mapbox 实验性 API，需 v3 样式（如 `standard`），未来版本可能变更。
::
