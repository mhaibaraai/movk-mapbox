## 简介

`MapboxTerrain` 调用 `setTerrain` 启用三维地形：`source` 缺省用 Mapbox 官方 `terrain-dem-v1`，`exaggeration` 控制夸张程度。倾斜视角下起伏最明显。

## 用法

调整 `exaggeration` 查看珠峰一带的地形起伏：

::component-example
---
name: TerrainExample
prettier: true
options:
  - name: exaggeration
    label: exaggeration
    default: 1.5
    items:
      - 1
      - 1.5
      - 2
---
::
