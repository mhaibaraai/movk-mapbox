## 简介

`useFeatureState` 在目标图层上自动维护 `feature-state` 的 hover / selected 状态：消费侧 paint 用 `['feature-state', 'hover' | 'selected']` 表达式驱动样式，无需重设 source 数据即可高亮悬浮与选中要素。返回当前悬浮 / 选中要素与清除选中方法。

::note{color="warning"}
要素必须带 `id`：在 source 上配置 `generateId: true` 或 `promoteId`，否则 `feature-state` 无法定位要素。
::

## API

### `useFeatureState()`{lang="ts-type"}

在目标图层上维护交互状态。

::field-group
  ::field{name="layerId" type="string" required}
  目标图层 id。
  ::
  ::field{name="options.mapId" type="string"}
  目标地图 id；在 `<MapboxMap>` 子树外使用时必填。
  ::
  ::field{name="options.hover" type="boolean"}
  维护 hover 状态，默认 `true`。
  ::
  ::field{name="options.select" type="boolean"}
  维护 click 选中状态，默认 `true`。
  ::
  ::field{name="options.cursor" type="boolean"}
  悬浮时设置 pointer 指针，默认 `true`。
  ::
::

返回 `UseFeatureStateReturn`：

::field-group
  ::field{name="hovered" type="ShallowRef<GeoJSONFeature | undefined>"}
  当前悬浮要素。
  ::
  ::field{name="selected" type="ShallowRef<GeoJSONFeature | undefined>"}
  当前选中要素。
  ::
  ::field{name="clearSelection" type="() => void"}
  清除选中。
  ::
::
