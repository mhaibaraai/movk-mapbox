## 简介

`useMapboxCamera` 封装常用相机操作：平滑飞行、缓动、立即跳转与缩放到范围。所有方法均返回 `Promise`，内部等待地图加载完成后执行，无需手动判断就绪状态。`fitBounds` 额外接受任意 GeoJSON，自动求其包围盒。

::note
在 `<MapboxMap>` 子树外使用时，经 `options.mapId` 指定目标地图。
::

## API

### `useMapboxCamera()`{lang="ts-type"}

创建相机操作助手。

::field-group
  ::field{name="options.mapId" type="string"}
  目标地图 id；在 `<MapboxMap>` 子树外使用时必填。
  ::
::

返回 `UseMapboxCameraReturn`：

::field-group
  ::field{name="flyTo" type="(options: EasingOptions) => Promise<void>"}
  平滑飞行到目标视角。
  ::
  ::field{name="easeTo" type="(options: EasingOptions) => Promise<void>"}
  缓动到目标视角。
  ::
  ::field{name="jumpTo" type="(options: CameraOptions) => Promise<void>"}
  立即跳转到目标视角。
  ::
  ::field{name="fitBounds" type="(target: LngLatBoundsLike | GeoJSON, options?: FitBoundsOptions) => Promise<void>"}
  缩放到目标范围；`target` 支持包围盒或任意 GeoJSON（自动求包围盒）。
  ::
::
