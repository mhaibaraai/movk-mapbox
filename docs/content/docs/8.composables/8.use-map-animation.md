## 简介

`useMapAnimation` 是地图帧动画原语：基于 `useRafFn`（组件卸载自动停止），仅在地图存在且样式就绪时调用 `frame` 回调，回调第二参为自启动起的累计毫秒数。动效组件与帧驱动图片均复用它。返回 `pause` / `resume` / `isActive` 控制运行状态。

::note
在 `<MapboxMap>` 子树外使用时经 `options.mapId` 指定目标地图。回调内每帧执行，避免在其中创建新对象或做重计算。
::

## API

### `useMapAnimation()`{lang="ts-type"}

创建帧动画循环。

::field-group
  ::field{name="frame" type="(map: Map, elapsedMs: number) => void" required}
  每帧回调；`elapsedMs` 为自启动起的累计毫秒数。
  ::
  ::field{name="options.mapId" type="string"}
  目标地图 id；在 `<MapboxMap>` 子树外使用时必填。
  ::
  ::field{name="options.immediate" type="boolean"}
  创建后立即启动，默认 `true`。
  ::
::

返回 `UseMapAnimationReturn`：

::field-group
  ::field{name="pause" type="() => void"}
  暂停动画。
  ::
  ::field{name="resume" type="() => void"}
  恢复动画。
  ::
  ::field{name="isActive" type="Readonly<Ref<boolean>>"}
  是否正在运行。
  ::
::
