## 简介

`defineMapboxControl` 用回调式 `onAdd` / `onRemove` 快速定义一个符合 `IControl` 接口的自定义控件，省去手写控件类的样板。`onAdd` 返回控件的 DOM 元素，`onRemove` 负责清理。返回的对象可直接传给 `map.addControl`。

::note
本助手只产出控件对象，不负责挂载。结合 [useMap](/docs/composables/use-map) / [useMapbox](/docs/composables/use-mapbox) 取得地图后调用 `addControl` 即可。
::

## API

### `defineMapboxControl()`{lang="ts-type"}

创建自定义控件对象。

::field-group
  ::field{name="onAdd" type="(map: Map) => HTMLElement" required}
  控件添加到地图时调用，返回控件的 DOM 元素。
  ::
  ::field{name="onRemove" type="(map: Map) => void" required}
  控件从地图移除时调用，负责清理。
  ::
::

返回 `IControl`：可直接传给 `map.addControl` 的控件对象。
