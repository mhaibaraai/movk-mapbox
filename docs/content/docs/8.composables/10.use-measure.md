## 简介

`useMeasure` 提供测距 / 测面交互：单击加点、移动预览、双击结束当前段、Esc 取消。它自管展示用的 geojson 源与图层，用 turf 实时计算并格式化读数。返回 `start` / `stop` / `clear` 控制方法与 `mode` / `active` / `result` 响应式状态。

::note
在 `<MapboxMap>` 子树外使用时经 `options.mapId` 指定目标地图。读数格式化复用 [formatDistance / formatArea](/docs/utils/measure)。
::

## API

### `useMeasure()`{lang="ts-type"}

创建测量工具。

::field-group
  ::field{name="options.mapId" type="string"}
  目标地图 id；在 `<MapboxMap>` 子树外使用时必填。
  ::
  ::field{name="options.color" type="string"}
  主色，默认 `'#f59e0b'`。
  ::
::

返回 `UseMeasureReturn`：

::field-group
  ::field{name="start" type="(mode: MeasureMode) => void"}
  进入测量：click 加点、mousemove 预览、dblclick 结束当前段。
  ::
  ::field{name="stop" type="() => void"}
  退出测量交互（保留已绘结果）。
  ::
  ::field{name="clear" type="() => void"}
  清空全部测量结果。
  ::
  ::field{name="mode" type="Ref<MeasureMode | undefined>"}
  当前测量模式。
  ::
  ::field{name="active" type="Ref<boolean>"}
  测量是否激活。
  ::
  ::field{name="result" type="Ref<string>"}
  当前段（含光标预览）的格式化读数。
  ::
::

### `MeasureMode`{lang="ts-type"}

测量模式：`'distance'` | `'area'`。
