## 简介

`useMapboxImage` 加载远程图片并注册为样式图片，供 symbol 图层经 `icon-image` 引用。`setStyle` 清空样式图片后会经 `onReady` 自动补回，组件卸载时移除。返回 `loaded` 状态便于在加载完成前给出占位反馈。

::note
设 `sdf: true` 注册为 SDF 图标后，可经 `icon-color` 着色。在 `<MapboxMap>` 子树外使用时经 `options.mapId` 指定目标地图。
::

## API

### `useMapboxImage()`{lang="ts-type"}

注册命名图片。

::field-group
  ::field{name="name" type="string" required}
  图片注册名称，供 `icon-image` 引用。
  ::
  ::field{name="url" type="string" required}
  图片 URL 地址。
  ::
  ::field{name="options.mapId" type="string"}
  目标地图 id；在 `<MapboxMap>` 子树外使用时必填。
  ::
  ::field{name="options.sdf" type="boolean"}
  作为 SDF 图标（可经 `icon-color` 着色）。
  ::
  ::field{name="options.pixelRatio" type="number"}
  像素密度，默认 `1`。
  ::
::

返回 `{ loaded: Ref<boolean> }`：图片是否已加载完成。
