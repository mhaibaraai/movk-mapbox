## 简介

`MapboxTooltip` 是 Popup 的 hover 模式：鼠标悬浮到 `layerId` 指定图层的要素上时跟随显示、移出即消失。默认插槽暴露命中要素（`feature`），可渲染其属性；按 `layerId` 委托到地图级监听，因此目标图层晚于本组件挂载也能命中。
