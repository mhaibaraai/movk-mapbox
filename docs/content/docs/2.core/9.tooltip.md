## 简介

`MapboxTooltip` 为 `layerId` 指定图层的要素挂载弹窗：默认 `trigger="hover"` 时跟随指针显示、移出即消失。默认插槽暴露命中要素（`feature`）与 `close`，可渲染其属性；按 `layerId` 委托到地图级监听，因此目标图层晚于本组件挂载也能命中。

`trigger` 支持 `hover` / `click` / `none`，其中 `none` 不绑定任何监听，可用于标绘等模式下临时停用。

prettier: true
collapse: true

::note
`click` 模式默认给出关闭按钮，但**不**启用 `closeOnClick` —— 否则弹窗会被开启它的那一次点击立即关闭。确有需要可经 `options` 覆盖。
::

::tip
组件内部只有一个 Popup 实例，因此同一时刻只展示一个要素的弹窗。若要让多个要素的弹窗同时常驻，见 [MapboxPopup](/docs/core/popup) 的「多弹窗共存」。
::
