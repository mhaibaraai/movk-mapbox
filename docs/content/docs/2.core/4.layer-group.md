## 简介

`MapboxLayerGroup` 把多个子 [MapboxLayer](/docs/core/layer) 归为一组：经 `beforeId` 提供统一的缺省插入锚点，经 `visible` 做组级显隐开关（组开关优先于子图层自身的 `layout.visibility`）。适合「一组相关图层一起显示/隐藏」的场景。

## 用法
