## 简介

`MapboxLayer` 是通用图层组件：`type` 选渲染方式，`paint` / `layout` / `filter` 控制样式，响应式变更走 `setPaintProperty` / `setLayoutProperty` / `setFilter` 增量下发，不重建图层。`source` 既可传 `sourceId` 引用 [MapboxSource](/docs/core/source)，也可传内联 source 对象（自动创建匿名源并随图层卸载）；`beforeId` 控制插入次序。
