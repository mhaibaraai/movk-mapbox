## 简介

`MapboxMarker` 在地图上放置一个标记：默认插槽提供自定义 DOM（省略则用 mapbox 默认水滴图标），位置经 `v-model:lnglat` 双向绑定。`options.draggable` 开启拖拽后，拖拽结束会把新坐标回写到绑定值。

## 用法

无插槽时使用默认标记，`v-model:lnglat` 控制位置：

::component-example
---
name: MarkerBasicExample
prettier: true
---
::
