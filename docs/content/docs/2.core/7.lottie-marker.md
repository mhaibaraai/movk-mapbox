## 简介

`MapboxLottieMarker` 在标记位置用 lottie-web 渲染矢量动画（基于 [MapboxMarker](/docs/core/marker) 的 DOM，而非 icon 帧驱动）。动画来源 `animationData`（内联 JSON）或 `path`（远程 URL）二选一，位置经 `v-model:lnglat` 绑定。
