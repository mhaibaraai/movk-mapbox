## 环境要求

::callout{color="info"}
- **Mapbox GL JS** —— v3（`mapbox-gl` 作为 peer 依赖）
- **Vue** —— 3.4+
- **Node.js** —— `^20.19.0 || >=22.12.0`
- 一个 **Mapbox access token**（[免费申请](https://account.mapbox.com/access-tokens/)）；如需天地图底图，另需天地图 `tk`
::

## 安装依赖

::code-group
```bash [pnpm]
pnpm add @movk/mapbox mapbox-gl
```

```bash [npm]
npm install @movk/mapbox mapbox-gl
```

```bash [yarn]
yarn add @movk/mapbox mapbox-gl
```
::

需要绘制能力（`MapboxDrawControl`）时，额外安装：

::code-group
```bash [pnpm]
pnpm add @mapbox/mapbox-gl-draw
```

```bash [npm]
npm install @mapbox/mapbox-gl-draw
```

```bash [yarn]
yarn add @mapbox/mapbox-gl-draw
```
::

## 选择接入方式

::card-group
  :::card{icon="i-simple-icons-nuxt" title="Nuxt 4" to="/docs/getting-started/nuxt"}
  作为 Nuxt 模块注册，组件与 composables 自动导入。
  :::

  :::card{icon="i-logos-vue" title="Vue + Vite" to="/docs/getting-started/vue"}
  经 Vite 插件 + vue-plugin，在纯 Vue 项目中使用同一套组件。
  :::
::
