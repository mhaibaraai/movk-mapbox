## 简介

`MapboxNavigationControl` 在地图上挂载缩放按钮与指北罗盘。`position` 指定停靠角（省略用地图默认位置），`options` 直通 mapbox-gl 的 `NavigationControl` 构造选项（`showZoom` / `showCompass` / `visualizePitch`）。控件在地图就绪后挂载，组件卸载时自动移除。

## 用法

调整 `position` 切换停靠角：

::component-example
---
name: NavigationControlExample
prettier: true
options:
  - name: position
    label: position
    default: top-right
    items:
      - top-left
      - top-right
      - bottom-left
      - bottom-right
---
::
