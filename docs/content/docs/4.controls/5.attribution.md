## 简介

`MapboxAttributionControl` 在地图角落展示数据来源署名。`options.compact` 折叠为可展开的图标按钮，`options.customAttribution` 追加自定义版权文本（字符串或字符串数组）。

::tip
样式或数据源自带的署名通常已自动显示；本控件用于自定义展示形态或补充版权。
::
