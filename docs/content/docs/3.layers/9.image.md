## 简介

`MapboxImageLayer` 把一张图片按四角经纬度（左上 / 右上 / 右下 / 左下）贴合到地图，常用于雷达图、扫描影像、历史地图等叠加。内部以 `image` 源 + `raster` 图层实现。
