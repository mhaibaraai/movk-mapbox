## 简介

`MapboxLayer` 设 `type="line"` 渲染线图层，`line-width` / `line-color` 控制描边，`line-cap` / `line-join` 控制端点与拐角样式。

::note{to="/docs/core/layer"}
line 由通用组件 MapboxLayer 承载，完整 Props 与事件见其文档。
::
