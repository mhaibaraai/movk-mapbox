## 简介

`MapboxLayer` 设 `type="fill"` 渲染面图层，`fill-color` / `fill-opacity` 控制样式，配合 `['get', ...]` 表达式可逐要素着色。

::note{to="/docs/core/layer"}
fill 由通用组件 MapboxLayer 承载，完整 Props 与事件见其文档。
::
