## 简介

`MapboxRasterLayer` 叠加外部栅格瓦片服务（`{z}/{x}/{y}` 模板 URL），支持 `tileSize` / `scheme` / `minzoom` / `maxzoom` / `attribution` / `opacity`。天地图等国内底图见 [扩展能力](/docs/extensions/tianditu)。
