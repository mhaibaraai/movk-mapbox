## 简介

`MapboxLayer` 设 `type="heatmap"` 把点密度渲染为热力图，`heatmap-radius` / `heatmap-intensity` / `heatmap-opacity` 调节观感。按属性加权的温度热力见 [MapboxTemperature](/docs/environment/temperature)。

::note{to="/docs/core/layer"}
heatmap 由通用组件 MapboxLayer 承载，完整 Props 与事件见其文档。
::
