## 简介

`MapboxLayer` 设 `type="fill-extrusion"` 把面挤出为 3D 体块，`fill-extrusion-height` 经 `['get', ...]` 读取属性逐要素设高；需给地图设 `pitch` 才能看出立体效果。3D 城市建筑见 [building](/docs/layers/building)。

::note{to="/docs/core/layer"}
fill-extrusion 由通用组件 MapboxLayer 承载，完整 Props 与事件见其文档。
::
