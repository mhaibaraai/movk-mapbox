## 简介

`MapboxClusterLayer` 把大量点数据自动聚合：内部用开启 `cluster` 的 geojson 源渲染「聚合圆 + 计数文字 + 散点」三层。点击聚合圆默认放大到展开级别（`autoExpand`），并派发 `clusterClick` / `pointClick` 事件。
