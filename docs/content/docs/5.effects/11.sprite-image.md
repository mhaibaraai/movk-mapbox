## 简介

`MapboxSpriteImage` 把一张雪碧图（行优先排布的多帧）切帧后注册为帧驱动图标，symbol 图层逐帧循环播放，零解码依赖；`frames` / `columns` / `rows` / `frameWidth` / `frameHeight` / `fps` / `size` 控制切帧与播放。
