## 简介

`MapboxDiffusionCircle` 在点要素上渲染扩散圆：多圈同心圆周期性向外扩张并渐隐；`color` / `maxRadius` / `duration` / `rings` / `opacity` 控制外观与节奏。

## 用法

切换主色查看扩散动画：

::component-example
---
name: DiffusionExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#3b82f6'
    items:
      - '#3b82f6'
      - '#10b981'
      - '#f59e0b'
      - '#f43f5e'
---
::
