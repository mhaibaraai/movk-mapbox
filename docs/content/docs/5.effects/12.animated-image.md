## 简介

`MapboxAnimatedImage` 用 `ImageDecoder` 把 GIF / APNG / WebP / AVIF 动图解码为帧，按各帧时长驱动 symbol 逐帧循环播放，零解码依赖；`size` 统一帧尺寸，`fps` 为无原生帧时长时的回退帧率。
