## 简介

`MapboxWaveCircle` 在点要素上渲染波浪圆：常驻实心底圆 + 周期向外扩张的描边波纹；`color` / `baseRadius` / `maxRadius` / `duration` / `rings` 控制外观与节奏。

## 用法

切换主色查看波纹扩散：

::component-example
---
name: WaveExample
prettier: true
clientOnly: true
options:
  - name: color
    label: color
    default: '#10b981'
    items:
      - '#10b981'
      - '#3b82f6'
      - '#f59e0b'
      - '#f43f5e'
---
::
