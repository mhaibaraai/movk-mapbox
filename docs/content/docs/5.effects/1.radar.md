## 简介

`MapboxRadar` 在点要素上渲染雷达扫描：canvas 生成扇形扫描贴图，symbol 图层按帧旋转；`color` / `radius` / `speed` 控制颜色、扫描半径与转速。

## 用法

调整 `speed`（度/秒）观察扫描转速：

::component-example
---
name: RadarExample
prettier: true
clientOnly: true
options:
  - name: speed
    label: speed
    default: 120
    items:
      - 60
      - 120
      - 240
---
::
