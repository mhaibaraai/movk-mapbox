## 简介

`MapboxWindowBuilding` 基于官方样式的 `composite/building` 渲染 3D 建筑，并用程序生成的窗户点阵贴图作 `fill-extrusion-pattern`；`rows` / `cols` / `litColor` / `darkColor` / `litRatio` / `seed` 控制窗格外观。需倾斜视角且缩放 ≥ `minzoom`（默认 15）。
