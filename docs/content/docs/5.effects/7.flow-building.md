## 简介

`MapboxFlowBuilding` 基于官方样式的 `composite/building` 渲染 3D 建筑：渐变着色 + 高亮带沿建筑高度循环流动；`color` / `flowColor` / `duration` / `bandHeight` / `maxHeight` 控制外观与节奏。需倾斜视角且缩放 ≥ `minzoom`（默认 15）。
